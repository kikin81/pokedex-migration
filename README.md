pokedex-migration
=================

pokedex-migration scripts in ruby

# Getting started

Download the sqlite database (25 mb) run

1. initialize submodule:

        $ git submodule init

2. update submodule (actual clone)

        $ git submodule update

3. gem dependencies
	Mongo
	$ gem update --system
	$ gem install mongo
	$ gem install bson_ext
	Sqlite3
	$ gem install sqlite3
